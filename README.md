# Instruções de execução

1. Instalar o Node:

> $ sudo apt update

> $ sudo apt install nodejs npm

2. Rodar os comandos:

> $ npm install
 
> $ npm rundev

OBS: instalando o npm com versões específicas: 

- Instalar o nvm para gerenciamento de versões:

> curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash

> source ~/.bashrc

- Instalando a versão 22:

> nvm install 22

> nvm use 22

> nvm alias default 22

- Verificando a versão instalada:

> node -v

> npm -v
